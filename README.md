# Tabib Hub

Student portal for the English-language medicine program at the Faculty of Medicine of Sousse.

## Run locally
```
npm install
npm run dev
```

## Deploy
This project is set up for Vercel. Connect this GitHub repo to Vercel — it will auto-detect Vite and deploy with zero configuration.

## Edit content
All course names, resources, and announcements live in `src/App.jsx` near the top of the file:
- `YEARS` — courses organized by year
- `RESOURCES` — summaries, past exams, guides
- `NOTICES` — announcement ticker
- `CATEGORIES` — feedback form categories
